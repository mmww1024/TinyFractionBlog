chrome的癖好

这篇想写chrome的一些特点，想到哪写哪，一点一点累计吧

一、chrome默认最小字体为12px

站在用户体验的角度，太小的字体对于视力不好的用户很不友好，所以任凭你设9px，10px的字体，chrome都默认显示为12px，但是有的时候小像素的字体不是给用户看的，只是起到装饰作用，比如下面这张图,英文单词只是起到了装饰作用，所以小一点也无妨嘛

在网上搜了下解决办法，可以用-webkit-text-size-adjust：none属性，但是Chrome27正式取消了此属性，有别的办法吗？
找到这个

.webkit {
font-size: 9px;
-webkit-transform: scale(0.75);
}

通过把div整个缩小，实现了字体的变小，12×0.75=9，对于其它浏览器来说，9px默认识别，仅需要对于webkit浏览器进行缩放。我在web端的ie,ff,opera上测试都通过了，移动端没有管，毕竟做的网页不是给移动端看的。

##二、chrome专有的hack##

针对Chrome和Safari等Webkit核心浏览器的CSS hack代码：

/* Webkit*/ 
@media screen and (-webkit-min-device-pixel-ratio:0) {.font1 {color:#f00} .border1 {border:1px solid #f00;} .bg3 {background:#f00;}}

同时，针对Opera浏览器的写法：

/*Opera*/ 
@media all and (-webkit-min-device-pixel-ratio:10000),not and all (-webkit-min-device-pixel-ratio:0) { .font1 {color:red;} }

针对Firefox浏览器的写法： 

/*Firefox*/
@-moz-document url-prefix() { .font1 {color:red} }
